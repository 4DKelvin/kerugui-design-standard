表单

基本实例

单独的表单控件会被自动赋予一些全局样式。所有设置了 .form-control 类的 <input>、<textarea> 和 <select> 元素都将被默认设置宽度属性为 width: 70%。所有 input [type="text"]元素高度设置为36px。 将 label 元素和前面提到的控件包裹在 .form-group 中可以获得最好的排列(margin-bottom: 28px)。

实例：

x 
<div class="form-group">
  <label><span class="text-warning">＊</span>我是输入框：</label>
  <input class="form-control" type="text" placeholder="我是默认文本">
  <span class="close-btn">x</span>
</div>

<label><span class="text-warning">＊</span>我是输入框：</label>
<input class="form-control" type="text" value="我是已输入文本">

<label><span class="text-warning">＊</span>我是单选框：</label>
<input id="radio-demo-1" type="radio" name="radio-demo"><label class="radio-inline" for="radio-demo-1">选项1</label>
<input id="radio-demo-2" type="radio" name="radio-demo"><label class="radio-inline" for="radio-demo-2">选项2</label>

禁用状态

为输入框设置 disabled 属性可以防止用户输入，并能对外观做一些修改，使其更直观。为输入框添加 .disabled 类，鼠标悬停效果变为不可点击。

当禁用单选框或复选框时，同时为对应的label标签加上 .disabled 类，label鼠标悬停效果变为不可点击。

实例：

<label><span class="text-warning">＊</span>我是输入框：</label>
<input class="form-control disabled" type="text" placeholder="我是不可点击文本" disabled>

<label><span class="text-warning">＊</span>我是单选框：</label>
<input id="radio-demo-1" type="radio" name="radio-demo"><label for="radio-demo-1">选项1</label>
<input id="radio-demo-2" type="radio" name="radio-demo" disabled><label class="disabled radio-inline" for="radio-demo-2">选项2</label>

下拉选择

为需要添加下拉选择样式的输入框添加一个 .dropdown-input 的 div 标签，为下拉图标添加 .dropdown-button 类，按钮样式不在此讨论。

实例：

<div class="form-group">
<label>我是输入框：</label>
<div class="dropdown-input">
  <input class="form-control" type="text" placeholder="我是可以下拉选择的输入框">
  <span class="caret dropdown-button"></span>
</div>

<div class="dropdown-input">
  <input class="form-control" type="text" placeholder="我是可以下拉选择的输入框">
  <button class="btn dropdown-button">按钮</button>
</div>

提示信息

为 .form-group 设置了属性 position: relative ，提示信息(.tips-info)和错误提示信息(.error)为绝对定位。在有提示信息的情况下，如果出错，错误提示信息覆盖提示信息。

实例中为了体现效果，额外给输入框元素加上错误样式。

P.S. 给textarea的label添加 .textarea-label 的类，确保标签文字与输入框上方对齐。

实例：

<label>&nbsp;&nbsp;&nbsp;&nbsp;我是输入框：</label>
<input id="checkbox-demo" type="checkbox" name="checkbox-demo"><label for="checkbox-demo">复选框</label>
<div class="form-group">
  <label class="invisible textarea-label">我是输入框：</label>
  <textarea class="form-control" name="textarea-demo" id="textarea-demo" rows="5"></textarea>
  <label class="tips-info">这是提示信息</label>
</div>

<label class="textarea-label">我是输入框：</label>
  <textarea class="form-control error" name="textarea-demo" id="textarea-demo" rows="5" placeholder="这是一个文本区域"></textarea>
  <label class="error">这是错误提示信息</label>

样式2

在 .multi-col-form 的块级元素下实现样式2的效果。用 .form-group-col 的块级元素将每一组label和input元素包裹起来（margin-bottom:28px）。

没有为input元素设置宽度。为了让input和label元素始终在同一列，可能需要额外的设置input的宽度和 .form-group-col的最小宽度。

按照规范图，将label宽度设置为122px，可能要按需更改。

实例：

<div class="multi-col-form">
  <div class="form-group">
    <div class="form-group-col">
    <label>我是标题</label>
      <div class="dropdown-input">
        <input type="text" placeholder="默认文字">
        <span class=" dropdown-button caret"></span>
      </div>
    </div>
    ...
</div>
